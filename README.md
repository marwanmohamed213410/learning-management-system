# learning management system

This project is a Java-based Learning Management System (LMS) that provides a comprehensive and user-friendly digital platform for managing various educational activities. Developed as a standalone application using core Java technologies, the system is designed to simulate the essential functionalities of modern online learning environments, focusing on simplicity, efficiency, and role-based access.

## System users

- Students:  who can follow their grades and assignments ,can upload them to the application and access their cources

## Project Purpose

- Demonstrate understanding of **OOP principles**: encapsulation, inheritance, polymorphism, abstraction.
- Practice **GUI development** with Java Swing.
- Apply **file handling** in Java for data persistence.

## Features

- Student login
- View list of available courses
- Submit assignments
- View grades and instructor feedback (mock data or file-based)
- File-based data storage (no external database)
- Built with Java Swing

## Technologies Used

- **Java SE**
- **Java Swing** (for GUI)
- **OOP concepts**: classes, inheritance, interfaces, encapsulation
- **File I/O** (`FileReader`, `FileWriter`, `ObjectInputStream`, etc.)
